How to use these sheets (the method that fixes recall)

  1. Read the concept box first. Each topic opens with the mental model — the why so facts stick.
  2. Then close it and retrieve. Each drill shows a question; the answer is hidden. Say or write your answer before clicking reveal. The struggle to recall is what builds memory — reading the answer does almost nothing.
  3. Mark misses, repeat tomorrow. The ones you fail are your real study list. Re-test them, don’t re-read them.
  4. Write the configs/code by hand. Your exam is handwritten — practice writing OSPF/Intent/Dart from a blank page, not just recognizing them.
Spacing beats cramming: three 40-minute retrieval sessions across three days beats one long read the night before. With ~12 days left, do one subject per day on rotation, then full mixed review in the last 3 days.

Export to PDF (handwriting practice / offline)

Each sheet’s Print / PDF button (in its sidebar) auto-reveals every answer and hides the controls. Or from Terminal:
